# Bilibiliwith163

Bilibiliwith163 是一个给 B 站直播间使用的弹幕点歌工具。

观众在弹幕里发送“点歌 歌名”，程序会搜索网易云音乐，把歌曲加入队列，并用本机播放器播放。OBS / 直播姬里只需要添加一个浏览器源，用来显示当前歌曲、队列和点歌结果。

音频由后端调用本机 `mpv` 播放，不依赖浏览器源出声，所以直播姬最小化、切到后台或 OBS 浏览器源被节流时，也能尽量保持播放稳定。

## 适合谁用

- B 站主播想让观众用弹幕点歌。
- 直播画面里想放一个可定制的点歌面板。
- 不想让 OBS / 直播姬浏览器源负责播放音频。
- 可以接受网易云歌曲能否播放取决于账号权限和接口返回。

## 下载安装

推荐使用 Releases 里的 Windows 压缩包：

```text
Bilibiliwith163-v版本号-windows-x64.zip
```

解压后会看到：

```text
Bilibiliwith163.exe
.env.example
README.md
RUNNING.md
RELEASE_NOTES.md
LICENSE
```

把 `.env.example` 复制一份，改名为 `.env`。

## 基础配置

打开 `.env`，至少修改这几项：

```text
PORT=3888
BILI_ROOM_ID=你的直播间号
BILI_COOKIE=你的B站Cookie
```

说明：

- `PORT` 是本地网页端口，默认 `3888`。
- `BILI_ROOM_ID` 是要监听的 B 站直播间号。
- `BILI_COOKIE` 用于连接直播间弹幕。
- 网易云不需要手动填写 Cookie，可以在控制台扫码登录。

## 启动

双击：

```text
Bilibiliwith163.exe
```

启动后打开：

```text
控制台：http://localhost:3888/dashboard.html
OBS源：http://localhost:3888/
```

如果你改过 `PORT`，把地址里的 `3888` 换成你的端口。

## 第一次使用

1. 打开控制台。
2. 确认房间号正确。
3. 在“网易云登录”区域扫码登录。
4. 如果提示没有本地播放器，点击“一键安装播放器”。
5. 在 OBS 或直播姬中添加浏览器源，地址填 `http://localhost:3888/`。
6. 让观众发送弹幕：

```text
点歌 歌名
```

例如：

```text
点歌 晴天 周杰伦
点歌 稻香
点播 bad apple
网易云 明天会更好
```

## OBS / 直播姬设置

添加浏览器源：

```text
http://localhost:3888/
```

建议：

- 开启透明背景。
- 根据直播画面裁剪或缩放点歌面板。
- 音频不从浏览器源播放，不需要给浏览器源单独开声音。
- 真正的声音来自后端调用的本机 `mpv`。

## 控制台功能

控制台地址：

```text
http://localhost:3888/dashboard.html
```

常用按钮：

- 暂停 / 继续
- 跳过
- 清空队列
- 停止
- 切换房间
- 网易云扫码登录
- 修改播放音质和缓存音质
- 编辑点歌面板外观

## 播放器

推荐使用 `mpv`。

如果电脑里已经有 `mpv.exe`，可以把它放进系统 PATH，或者在 `.env` 中填写完整路径：

```text
LOCAL_PLAYER_PATH=D:\path\to\mpv.exe
```

如果没有播放器，控制台会显示“一键安装播放器”。点击后会下载开源便携版 `mpv` 到：

```text
.cache/player/mpv/
```

这个安装不会修改系统 PATH。

## 音质设置

控制台里有两项：

- 播放音质：直播时即时播放使用，建议保持“标准”，更稳。
- 缓存音质：后台缓存文件使用，可按网络情况调整。

如果直播时更看重流畅度，优先降低播放音质。

## 文件和缓存

运行后会生成：

```text
.env
.cache/
```

其中：

- `.env` 保存你的房间号、B 站 Cookie、网易云登录状态等配置。
- `.cache/audio/` 保存音频缓存。
- `.cache/player/` 保存便携版播放器。
- `.cache/appearance.json` 保存外观配置。

不要把 `.env` 发给别人。

## 常见问题

### 控制台打不开

先确认程序窗口还在运行，再检查 `.env` 里的端口。

如果端口是：

```text
PORT=3888
```

控制台就是：

```text
http://localhost:3888/dashboard.html
```

### OBS 源没有声音

这是正常的。浏览器源只显示画面，不负责播放声音。

声音由本机播放器输出。请检查：

- 控制台是否显示本地播放器已就绪。
- 系统音量是否正常。
- `mpv` 是否安装成功。

### 歌曲搜得到但播不了

可能原因：

- 网易云账号没有播放权限。
- 歌曲本身不可用。
- 网易云接口没有返回可播放地址。
- 当前网络访问网易云不稳定。

可以先在控制台扫码登录网易云，再重新点歌。

### 后台播放卡顿

这个版本已经把音频播放移到后端本地播放器，正常情况下比浏览器源播放更稳定。

如果仍然卡顿：

- 确认播放器后端是 `mpv`。
- 降低播放音质。
- 确认电脑没有把 `mpv.exe` 或本程序限制到节能模式。

### 一键安装播放器失败

安装器需要访问 GitHub 下载便携版 `mpv`。如果失败，可以手动下载 Windows x64 版 `mpv`，然后在 `.env` 里填写：

```text
LOCAL_PLAYER_PATH=你的mpv.exe完整路径
```

## 安全提醒

- `.env` 里可能有 B 站和网易云登录信息。
- 不要把 `.env`、`.cache/`、日志文件发给别人。
- 本项目不会绕过平台权限、会员限制或版权限制。
- 歌曲是否可播，取决于网易云返回结果和你的账号权限。

## 从源码运行

如果你不使用 Release 包，也可以用源码运行：

```powershell
npm install
copy .env.example .env
npm start
```

需要 Node.js 18 或更高版本。
